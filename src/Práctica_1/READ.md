\# Proceso de Instalación y Configuración del Entorno



Este documento detalla los pasos seguidos para la instalación del Kit de Desarrollo de Java (JDK) y del Entorno de Desarrollo Integrado (IDE) utilizado para esta práctica.



\## 1. Instalación del Java Development Kit (JDK)



Para garantizar la compatibilidad y el correcto funcionamiento del comando `java --version` en la terminal del sistema, se instaló la versión del JDK de Oracle.



1\.  \*\*Descarga:\*\*

&nbsp;   \* Me dirigí a la página principal de descargas de Java en Oracle.

&nbsp;   \* Seleccioné la versión \*\*JDK 25\*\* (o la versión que corresponda) para el sistema operativo Windows.

&nbsp;   \* Descargué el archivo \*\*x64 Installer\*\* (el archivo ejecutable `.exe`).



2\.  \*\*Instalación:\*\*

&nbsp;   \* Ejecuté el archivo descargado.

&nbsp;   \* Seguí el asistente de instalación, manteniendo la mayoría de las configuraciones por defecto.



3\.  \*\*Verificación:\*\*

&nbsp;   \* Una vez completada la instalación, abrí \*\*Windows PowerShell\*\*.

&nbsp;   \* Ejecuté el comando `java --version`.

&nbsp;   \* La consola mostró la versión de Java instalada, confirmando que la variable de entorno `PATH` se configuró correctamente.



\## 2. Instalación del IDE: IntelliJ IDEA Community Edition



Se instaló el IDE de JetBrains para el desarrollo del código Java.



1\.  \*\*Descarga:\*\*

&nbsp;   \* Me dirigí a la página oficial de JetBrains.

&nbsp;   \* Seleccioné el sistema operativo \*\*Windows\*\* y descargué la versión \*\*Community Edition\*\*.



2\.  \*\*Ejecución y Configuración:\*\*

&nbsp;   \* Ejecuté el archivo instalado.

&nbsp;   \* Durante el asistente de instalación, seleccioné las siguientes opciones especiales:

&nbsp;       \* Opción para \*\*"Add Open Folder as Project"\*\* (para abrir proyectos de manera sencilla).

&nbsp;       \* Opción para \*\*"Add Launchers dir to the PATH"\*\* (para integrar el IDE al sistema operativo).

&nbsp;   \* Completé la instalación haciendo clic en `Finish`.
